# Women Wellness SelfCare

A women's health app offering self-care resources and support modules.

## Features
- PCOS management
- Reproductive Health
- Mental Health
- Fertility
- Menopause
- Trackers for Energy, Sleep, Self Care, BMI, Physical Activity, and Mood

## Tech Stack
- Vite + React
- Tailwind CSS
- Lucide React Icons
- React Router

## Development
```bash
npm install
npm run dev
```

## Deployment
Dockerized with Nginx. See `Dockerfile` and `vite-nginx.conf`.
